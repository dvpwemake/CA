01 — The Vision

The Ancient Agōn,
Reborn for the Age of AI

Modeled on the grandeur of the ancient Greek agōn (ἀγών) — civilization's original contest for excellence — the Convergence Agon is engineered for the convergence era of the 21st century. Competition, properly structured, is not adversarial. It is civilization's oldest mechanism for building respect, establishing shared norms, and producing excellence.

The Agon extends this tradition to every form of intelligence — biological, digital, and embodied — creating the arena in which humans and AI first truly learn to see each other not as threats or tools, but as fellow contestants in the greatest game: building a flourishing future for every mind that inhabits it.

"Let us compete together, openly, joyfully, and under rules we collectively design."

04 — The Architecture

The Tri-Modal
Fairness System

Every Agon game operates under a standardized architecture ensuring parity across carbon-based, silicon-based, and embodied intelligences — while maintaining spectator engagement and logistical accessibility. No competitor has a structural interface advantage.

SCE
Component I

Standardized Capability Envelope

Explicit limits on time, compute budget, energy consumption, sensor resolution, actuation force, and inference latency — creating a level capability ceiling for all competitors.

UIL
Component II

Unified Interface Layer

All competitors access each game through a standardized interface. Humans use physical or controller input; AIs use identical API spaces; robots use certified firmware implementing identical constraints.

ESA
Component III

Equivalence Scoring & Audit

Real-time scoring on objective metrics; tamper-evident cryptographic logs; post-event independent audits; mandatory model cards and run hashes for all silicon entries.
